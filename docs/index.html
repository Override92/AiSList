Features

Why AiSList & AiBlock?

A simple but powerful solution to clean up your YouTube recommendations

Community-Maintained

The blocklist is curated and updated by the community. Anyone can suggest channels to add through GitHub issues or Discord.

Real-Time Blocking

The extension hides AI slop channels instantly as you browse YouTube. No page reloads needed - content disappears automatically.

Auto-Updates

The blocklist syncs automatically from GitHub. New AI channels get blocked without you having to do anything.

Privacy-First

No tracking, no analytics, no data collection. The extension only reads the blocklist and filters your feed locally.

Open Source

100% open source and transparent. Review the code or contribute improvements.

Cross-Browser

Works on both Firefox and Chrome. Same great experience regardless of your browser preference.

Contribute

Help Grow the List

Found an AI slop channel? Here's how you can help expand the blocklist

1

Open a GitHub Issue

Head to the AiSList repository and create a new issue. Include the channel URL and a brief explanation of why it should be blocked.

2

Submit a Pull Request

Fork the repo, add the channel ID to aislist_blocklist.txt, and submit a PR. Each line is one channel ID - easy to contribute!

3

Report on Discord

Not familiar with GitHub? No problem! Join our Discord server and report AI channels there. Community members will review and add them to the list.
